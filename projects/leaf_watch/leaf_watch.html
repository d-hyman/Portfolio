Project Overview

Large scale deforestation accelerates soil errosion, habitat loss, and intensifies climate change. This can come from direct human activity such as forest clearing, or from wildfires intensified by climate change. Desbite global reforestation initiatives, trees are rarely replanted in areas which require them the most, leading to drastic differences in deforestation rates around the world. Today, there is no public, location-specific tool to see where this decline is most severe, which results in trees being planted in unncessary locations.

Introducing Leaf Watch, a web application that allows users to visualize deforestation rates around the world using satellite data from NASA. Producing, targeted, data-driven reforestation—getting the right trees into the right places at the right time for the biggest ecological impact.

Key Features

  • Feature 1: Explore NDVI and thermal data on an interactive Map
  • Feature 2: View a timelapse or scoll through past years to see changes over time
  • Feature 3: Utlize a Machine Learning model that predicts future trends based on historoical data
  • Feature 4: Analyze detailed charts and visualizations of past and predicted metrics
  • Feature 5: Converse with an integrated AI Assistant

Technology Stack

  • Frontend: JavaScript, React, React-Dom library, Leaflet.js
  • Backend: Python, Node.js, FastAPI
  • APIs: NASA API, React-Dom APIs

How to run locally via terminal

  • Install node.js: install npm
  • Create Python Enviornment: python -m venv venv
  • Enter backend directory: cd backend
  • Install server via FastAPI: install fastAPI uvicorn
  • Run main backend file: uvicorn main:app --reload
  • Run website locally: npm run dev
